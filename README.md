# Ogre
## Description
This is a firefox extension with the goal of making my experience with YouTube a little less distracting and a little more productive and focussed. To this end, I am creating an extension that removes
the YouTube home page, removes recommendations while watching videos, removes any short form content. This alows for an experience wherin I am able to just follow the channels I'd like to follow and 
removes the tempation for short form content or the endless consumption of videos I dont actually care about.
